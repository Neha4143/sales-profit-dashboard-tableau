# Sales & Profit Dashboard

## Project Overview

This Tableau dashboard analyzes sales and profit performance across different product categories, sub-categories, states, and time periods using the Sample Superstore dataset.

## Tools Used

- Tableau Public
- Sample Superstore Dataset

## Key KPIs

| KPI | Value |
|------|------|
| Total Sales | 2,297,201 |
| Total Profit | 286,397 |
| Profit Ratio | 12.47% |

## Dashboard Features

- Sales & Profit by Sub-Category
- Profit by State Analysis
- Sales Trend Over Time
- Profit Trend Over Time
- Category-wise Sales vs Profit Comparison

## Business Insights

1. Technology generated the highest sales among all categories.
2. Profit increased consistently from 2014 to 2017.
3. Some states generated negative profit despite strong sales.
4. Phones and Chairs were among the top-performing sub-categories.
5. Profitability varied significantly across product categories.
